# ProjetoCadastroED2

## Objetivo geral

Criar um sistema de cadastro que armazena senhas de forma segura utilizando uma função hash criptográfica 

## Funcionalidades
- Criar novas contas e armazena-las na memória ou até mesmo em disco
- Visualizar o resultado das funções criptográficas da sua possível senha

## Tecnologias
- O programa sera feito 100% em c utilizando de conceitos de Tabela Hash, Funções Hash Criptográficas, funções de busca e armazenamento e manipulação de arquivos .txt
